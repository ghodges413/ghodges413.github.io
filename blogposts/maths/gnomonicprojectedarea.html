Gnomonic Projected Area

2018/12/30

TL;DR

Gnomonic Projection is the projection of the plane, normal to the z-axis located at \( z = 1 \), onto the unit sphere.

The area of the first quadrant of the plane with extents \( [0,x] \cup [0,y] \) is given by: $$ A( x, y ) = \tan^{-1}{ \frac{ x \cdot y }{ \sqrt{ x^2 + y^2 + 1 } } } $$

Derivation

The projection of a point in the plane \( r( x, y, 1 ) \) to a point on the sphere \( s( x, y, z ) \) is given by: $$ \vec{ s }( x, y, z ) = \frac{ \vec{ r } }{ \sqrt{ \vec{ r } \cdot \vec{ r } } } $$ And surface area is given by the double integral: $$ A = \int{ \int{ \parallel \frac{ \partial{ r } }{ \partial{ x } } \times \frac{ \partial{ r } }{ \partial{ y } } \parallel dxdy } } $$ So knowing that the components of \( \vec{ s } \) are given by: $$ s_x = \frac{ x }{ \sqrt{ x^2 + y^2 + 1 } } $$ $$ s_y = \frac{ y }{ \sqrt{ x^2 + y^2 + 1 } } $$ $$ s_z = \frac{ 1 }{ \sqrt{ x^2 + y^2 + 1 } } $$ Then this means that the partial derivates are given by: $$ \frac{ \partial s }{ \partial x } = \begin{bmatrix} \frac{ 1 - x^2 }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } }\\ \frac{ -xy }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } }\\ \frac{ -x }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } }\\ \end{bmatrix}, \frac{ \partial s }{ \partial y } = \begin{bmatrix} \frac{ -xy }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } }\\ \frac{ 1 - y^2 }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } }\\ \frac{ -y }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } }\\ \end{bmatrix}, \frac{ \partial s }{ \partial z } = \begin{bmatrix} 0\\ 0\\ 0\\ \end{bmatrix} $$ And performing the cross product yields: $$ \frac{ \partial s }{ \partial x } \times \frac{ \partial s }{ \partial y } = \begin{bmatrix} \frac{ x }{ ( x^2 + y^2 + 1 )^2 }\\ \frac{ y }{ ( x^2 + y^2 + 1 )^2 }\\ \frac{ 1 }{ ( x^2 + y^2 + 1 )^2 }\\ \end{bmatrix} $$ And then taking the inner product of this vector with itself gives: $$ \parallel \frac{ \partial s }{ \partial x } \times \frac{ \partial s }{ \partial y } \parallel = \frac{ 1 }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } } $$ Now we can plug this into the above integral to get the equation for the area: $$ A = \int{ \int{ \frac{ 1 }{ ( x^2 + y^2 + 1 )^{ \frac{ 3 }{ 2 } } } dxdy } } $$ Isn't that simple?
